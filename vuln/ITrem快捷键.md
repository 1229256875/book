新建标签：command + t

关闭标签：command + w

切换标签：command + 数字 或者 command + 左右方向键

切换全屏：command + enter

查找：command + f

垂直分屏：command + d

水平分屏：command + shift + d

切换屏幕：command + option + 方向键 

查看历史命令：方向键上键;

清除当前行：ctrl + u 

到行首：ctrl + a

到行尾：ctrl + e

前进后退：ctrl + f/b (相当于左右方向键)

清屏1：command + r

清屏2：ctrl + l
